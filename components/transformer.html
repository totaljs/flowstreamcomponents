With this component you can easily modify incoming data. Then the `data` content is delivered. For storing temporary/persistent data you can use `repo {Object}` variable. __Example 1__: ```js // "data" is a reference to message data data.name = data.name.toUpperCase(); ``` __Example 2__: ```js data = data.toUpperCase(); ``` __Example 3__: ```js var tmp = data; data = {}; data.name = tmp.Name; ```
