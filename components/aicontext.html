This component can load multiple Markdown contexts for the LLM. It replaces text phrases defined in the header, such as `[filename_defined_in_the_header]` with the content defined in the context. __Context example__: ```md ---------------------------------------------------------------- Name: document1.md ---------------------------------------------------------------- MARKDOWN 1 ---------------------------------------------------------------- Name: document2.md ---------------------------------------------------------------- MARKDOWN 2 ``` ## Usage ``` [document1.md] ``` __Output__: ``` MARKDOWN 1 ```
