Uses mquery under the hood. Here is the [documentation](https://www.npmjs.com/package/mquery) ## Input Example ```javascript { database: 'database-name', collection: 'collection-name', data: // this prop will be passed into the query function } ``` ## Query editor ```js // query; mquery instance // data; data from the incomming object query.find(); // return all documents ``` ## Important In the `.select()` you cannot use inclusion and exclusion together. This is invalid: `.select('firstname -lastname')` Valid exclusion: `.select('-lastname')` or `.select('-email -lastname')` etc. Valid inclusion: `.select('name email')` or `.select('email lastname')` etc.
